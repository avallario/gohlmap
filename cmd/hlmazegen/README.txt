To use hlmazegen:

Create a directory containing the following:
hlmazegen.exe
cell.map, empty.map
hlcsg.exe, hlbsp.exe, and hlvis.exe (these can be acquired from http://zhlt.info/download-zhlt.html)

Edit empty.map: in the empty quotes next to "wad", enter the path to a copy of halflife.wad

Run hlmazegen.exe
